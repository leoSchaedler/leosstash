O que é Sk8Hub?

Sk8Hub é um HUB (Central) de vídeos de skate com seu player próprio. O App proporciona uma experiência fullscreen para os videos, assim como a opção de escolha de qual vídeo irá ser rodado. No Hub existe uma música ambiente que pode ser desligada à preferência.

Sobre o desenvolvimento

Sk8Hub foi o segundo projeto da matéria de Experiência Criativa, após a experiência adquirida com o primeiro projeto, os orientadores propuseram uma nova ferramenta, chamada "Processing", ferramenta que foi criada para artistas aprenderem a programar, pois possui várias funções pré-definidas para desenho e desenvolvimento gráfico. O Processing roda em Java, mas pode ser configurado para Python, durante o projeto utilizei a versão em Java. Foi muito interessante explorar a ferramenta e descobrir do que ela é capaz, mas também foi um desafio, já que na primeira atividade não utilizamos nenhum tipo de código, e aqui o Processing necessita de códigos coerente para funcionar bem. Meu projeto foi bem sucedido graças a ajuda de bibliotecas disponíveis dentro do programa e muita persistência.
